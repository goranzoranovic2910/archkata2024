## Administrator Endpoint Methods

```
PUT /api/admin/updateCandidateHireStatus: update the hire status of a candidate
PUT /api/admin/updateUser: update general user data, including candidates and employers
PUT /api/admin/updateCandidate: update specific candidate-related data (e.g., skills, resume)
PUT /api/admin/updateEmployer: update employer-related data (e.g., company details, job posts).
PUT /api/admin/updateMatchingStrategyConfig: update the matching strategy configuration for the skill-matching service (Cosine Similarity vs. LLM).
```
